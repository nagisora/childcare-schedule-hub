## 解決したいこと（子育て拠点のイベントスケジュール確認が大変）

私は1年の育休を取り、子育てを頑張っていますが、家での育児は大変なので地域の子育て拠点を活用させていただいています。

子育て拠点とは、赤ちゃんと保護者が過ごせる場所で、色んなおもちゃがあって保育士さんが常駐していたりと、子育てを支援してくれる行政サービス拠点です。

その子育て拠点は「好きな時間に来て、好きな時間に帰って良い」という、フリータイムで開放されているのですが、子育て講座などのイベントがあると予約したイベント参加者しか拠点に入れません。せっかく行っても入れずに帰らなければならないことがあります。

それを防ぐため、子育て拠点はイベントスケジュール（カレンダー）を公開してくれていますが、主にInstagramの投稿であったり、独自サイトでの公開もあります。

それだと、いちいち色んな所に見に行かなければならないので非常に手間で、見ずに拠点に突撃する方が多く、トンボ帰りされている方をよく見ます。
実際、私もこのアプリを作る前は同じ経験を何度もしました。

アプリを作る前は、毎回こんな手順でした。

1. Instagramを開く
2. 検索欄で行きたい拠点の名前を入力し検索
3. 拠点アカウントの投稿の中から「月間スケジュール」の投稿を探す
4. スケジュール投稿が見れる

1つの拠点だけならこれくらいの手間は仕方ないかと諦められますが、複数の拠点に行くような私の場合だと、色々な拠点のスケジュールを探し当てるのが非常に面倒なのです！！
施設を検索してスケジュール投稿を探す必要があるし、探しているうちに前の施設のスケジュールが何だったか忘れます。

一方、私が作ったアプリだと、

1. アプリを開くだけで、スケジュール投稿が見られる

ので、ストレスフリーです！

## アプリ概要（MVP時点）

子育て拠点のイベントスケジュール（カレンダー）を、月ごとに一覧で見られるWebアプリです。

個人開発アプリはリリースするのが大切だと思うので、最低限の機能を持たせてリリースしようと思っています。

- 名古屋市（愛知県）の子育て拠点のみ。私の居住地
- 月間スケジュールを一覧で見られる

冒頭にも書きましたが、最低限はできていて自分が使う分には問題ない状況です。

ただ、子育て拠点のInstagramのURLとスケジュール投稿を自動取得させることが難しく、結構手間取っています。

とりあえず、年内というリミットを設けてできた所までで公開できればと思っています。

## オープンソースで公開

このアプリは以下のGitHubリポジトリでオープンソースとして公開しています。

https://github.com/nagisora/childcare-schedule-hub

オープンソースにした理由は以下のとおりです。

- 良い情報はみんなで共有して、人類が良くなるようにしたい
- このAIが発達した時代に、ソースコードをクローズにしてもあまり意味ない
- 私がこのアプリに飽きて放置した場合に、引き継ぎたい方がいれば引き継いでほしい
- 私がIT政党「チームみらい」に共感しサポーターもしているので、オープンソースにしたい

ちなみに、端的にこのアプリの肝は、

**Instagram投稿の自動取得**

なので、それをやりたい方は参考になるかもしれません。
