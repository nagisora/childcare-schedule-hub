# 開発セッション（dev-sessions）

このディレクトリは、**必要なときだけ**開発セッション記録を作るための最小運用ディレクトリです。

## このディレクトリに残すもの

- `template-session.md`: セッション記録テンプレート
- `session-definition.md`: このプロジェクトにおける「セッション」の定義

## 運用方針（最小運用）

- 過去の大量ログ（`2025/**` など）は軽量化方針により整理済みです。
- 記録は「毎回必須」ではなく、以下のような場合に限定します。
  - 方針判断の経緯を残したい
  - 後日再実行が必要な運用手順を残したい
  - 失敗ケースや品質チェック結果を再利用したい
- 恒久仕様は `docs/01-06` や `docs/05-*` を正本とし、本ディレクトリは補助ログとして扱います。

## 新規作成ルール（必要時のみ）

- 保存先: `docs/dev-sessions/YYYY/MM/`
- ファイル名: `YYYYMMDD-連番-説明.md`
- テンプレート: `template-session.md` をコピーして利用
